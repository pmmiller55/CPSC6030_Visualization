Current Analyics in Electrical Engineering

                                                                                                                               

Our paper discusses visualization techniques that allow for proper data comparison that highlights trends and anomilies in power grid design for real world application.
If we wanted to identify where the highest voltage or wattage is being pulled, you cant identify that information quickly.
Data is usually compiled with Microsoft Excel and includes several columns with thousands of rows.
In current real world application, the nodes are arranged in order of typical power flow, which means the links and nodes are static and provide no information.
Modeling our visualization off the techniques used in the paper linked above, the links are changing length based on fifteen minute current (A) values. The higher the current, the longer the link will be.
To emphasize the relationships between the nodes (and for the purposes of testing the code while writing it), the link length, colors, and node size all represent the current flowing between the nodes.
Other data options are real power (kV) and reactive power (kVA) which have also been collected at the same time stamps. Utilizing these values together in a node graph allows us to study more trends and can show any issues or need for future construction.
