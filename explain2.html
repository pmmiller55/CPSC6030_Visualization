Interactivity Example 1 - Watts and Volts

Set the edge to Watts using the drop down menu, and set the color to Volts. When the length of the edge increases, the Watts are increasing indicating someone is using more power. When the color is green, the voltage is running the same average as the other buses. Yellow means the bus is above average. Red is below average. When the watts are the same for each bus we expect similar voltage. Use the slider to change the date to ++++++++. Notice that one of the buses has a voltage that dropped below the average of the others indicated by the red color. However, the watts are still the same length between the buses. This could indicate that someone is illegally bypassing the meter.
